Font Uses:

Heading

Protest Riot is used for main heading
h1: font size:32px font weight:normal line height:normal letter spacing:1px
h2: font size:28px font weight:normal line height:1.2 letter spacing:1px
h3: font size:24px font weight:normal line height:1.2 letter spacing:1px
h4: font size:20px font weight:normal line height:1.2 letter spacing:1px

Paragraphs

Roboto font is used for all paragraphs with normal or medium font weight, font size: 20px or 18px, line height:1.5 or normal and line spacing:1px or normal

Captions or other text

Roboto with italic weight is used for all captions , font size: 16px, letter spacing small

CTA's

Roboto with bold weight is used for all CTA'S, font size:24px letter spacing: 2px
